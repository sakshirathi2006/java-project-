# Problem Statement
Managing student courses and enrollments manually is time-consuming and error-prone. CCRM automates this process.

# Scope
Java console app for student/course management with enrollment rules and GPA calculation.

# Target Users
University administrators and faculty.

# High-level Features
- Student and course CRUD operations
- Enrollment with credit limit validation  
- GPA calculation and transcripts
- CSV import/export and backups
